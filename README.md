# uni-file-sorter

Automatically sorts downloaded university PDFs into correct module folders, based on file contents. 

## Features
- Monitors the downloads folder for new PDFs.
- Extracts text from PDFs.
- Sorts PDFs to the module folders using keywords.
- Places unclear PDFs into unsorted folder.

## MVP 
- Recognize PDFs in downloads folder.
- Extract text.
- Determine right module folder, by rules.
- Move PDF.

## Status
- Current version works.
- Still work in progress.

## Run 
- Adjust the paths in `AppConfig.kt`.
- Run the project.

## Notes
- Invalid PDFs are ignored.
- Sorts existing PDFs too.
